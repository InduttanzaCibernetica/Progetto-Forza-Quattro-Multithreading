# Progetto Forza Quattro Sincronizzata
Applicazione del classico giodo Forza Quattro in java

### Requisiti
* **JDK 11+**
* **Maven 3.8+**

## Esecuzione:

## Server
* Accedere alla cartella dove si trova il server -> \Progetto-Forza-Quattro-Multithreading\ForzaQuattroServer
* Eseguire il comando mvn -q exec:java -Dexec.mainClass=forza4.server.Server

## Client
* Da terminale accedere alla cartella dove si trova il server -> \Progetto-Forza-Quattro-Multithreading\ForzaQuattroClient
* Eseguire il comando mvn -q exec:java -Dexec.mainClass=forza4.client.Client
* Inserisci l'IP del server per potersi connettere
